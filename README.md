# 송파구 토지이용 분석 시스템

송파구 필지 데이터 및 집계구 인구통계를 시각화하는 React + Leaflet 웹 애플리케이션입니다!

##  실행 방법

```bash
npm install
npm run dev
```
→ http://localhost:5173 에서 실행

##  기술 스택

- **React** (Vite)
- **Leaflet.js** — 지도 렌더링
- **Recharts** — 파이차트, 바차트
- **브이월드 WMTS API** — 베이스맵

##  데이터 출처

| 파일 | 출처 | 설명 |
|------|------|------|
| parcels.geojson | 브이월드 연속지적도 | 송파구 31,153개 필지 경계 + 속성 |
| haengjeong.geojson | 브이월드 행정구역 | 송파구 13개 행정동 경계 |
| yongdo.geojson | 브이월드 토지이용계획정보 | 용도지역 폴리곤 |
| jipgegu_stats.json | 통계지리서비스(SGIS) | 2024년 집계구 인구/가구 통계 |

##  주요 기능

### 좌측 패널
- 레이어 토글: 행정동 경계 / 집계구 통계 / 필지
- 필지 색상 모드: 건축물 주용도 / 용도지역 / 지목
- 투명도 슬라이더
- 범례 / 집계구 choropleth 범례

### 우측 통계 패널 (4개 탭)
- **건축물 주용도** — 파이차트 + 필지수/연면적 테이블
- **용도지역** — 파이차트 + 필지수/연면적 테이블
- **지목** — 파이차트 + 필지수/면적 테이블
- **집계구 통계** — 행정동별 인구/가구/연령 분석
  - 행정동 선택 → 요약 카드 6개
  - 연령별 인구 바차트 (남/여 구분)
  - 행정동 비교 바차트 (지표 선택)
  - 전체 행정동 통계 테이블

### 지도 인터랙션
- **필지 클릭** — 필지번호 / 용도지역 / 주용도 / 지목 / 면적 팝업
- **집계구 레이어** — choropleth (총인구/인구밀도/평균나이/노령화지수/가구수 선택)
- **행정동 클릭** — 집계구 모드에서 인구통계 팝업
- 행정동 이름 라벨 토글
